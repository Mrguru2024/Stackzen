# Select

A select component that allows users to choose an option from a dropdown list.

## Features

- ✅ Accessible by default
- 🎨 Customizable styling with Tailwind CSS
- 🌙 Dark mode support
- ⌨️ Keyboard navigation
- 📱 Responsive design
- ♿ ARIA attributes support
- 🔍 Search functionality
- 📦 Group options support

## Usage

```tsx
import {
  Select,
  SelectContent,
  SelectItem,
  SelectTrigger,
  SelectValue,
} from '@/components/ui/select';

export default function Example() {
  return (
    <Select>
      <SelectTrigger className="w-[180px]">
        <SelectValue placeholder="Select a fruit" />
      </SelectTrigger>
      <SelectContent>
        <SelectItem value="apple">Apple</SelectItem>
        <SelectItem value="banana">Banana</SelectItem>
        <SelectItem value="orange">Orange</SelectItem>
      </SelectContent>
    </Select>
  );
}
```

## Accessibility

The Select component follows WAI-ARIA best practices:

- Uses native select input for better accessibility
- Supports keyboard navigation
- Includes proper ARIA attributes
- Maintains focus states for keyboard users
- Provides clear visual feedback for different states
- Supports screen readers

## Best Practices

1. Always provide a placeholder or label
2. Use clear and concise option labels
3. Group related options together
4. Consider using a description for complex selections
5. Maintain proper spacing and sizing
6. Use appropriate error states when validation fails
7. Consider adding search functionality for long lists
8. Use icons to enhance visual understanding
